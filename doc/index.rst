Welcome to Lazuli's documentation!
==================================

You are reading the documentation of the Lazuli project.

This documentation is divided in two parts: **User** and **Kernel**.

The **User** documentation is intended for end users, who wish to develop
real-time systems and applications using the Lazuli kernel.
This documentation will guide you through the process of getting Lazuli sources,
setting up a working development environment, and compiling your code with
Lazuli. It will also give you detailed information about the functionalities
provided by Lazuli.

The last chapter of this documentation is the **Kernel** documentation.
It is intended for programmers, who wish to code in the Lazuli kernel, hacking
and improving it.
This documentation will give you detailled information about the inner working
of Lazuli.
